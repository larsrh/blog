XML Messaging

Stefan Tilkov, Oct 21, 2005

Comments to Wednesday’s entry about XML messaging show not everybody agrees. I remain convinced that if you don’t derive any additional value from using XML, if it simply remains an implementation detail, you’re better off using something else. In some ways, this is similar to using HTTP as a mere transport for SOAP — while it’s possible to use HTTP this way, you might just as well be doing SOAP over TCP directly.
