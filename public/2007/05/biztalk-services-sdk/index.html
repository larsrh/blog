BizTalk Services SDK

Stefan Tilkov, May 4, 2007

Steve Maine about BizTalk Services SDK:

The feature set in this SDK is really quite amazing. Using these bits, you can do simple HTTP GET’s on URI’s. You can syndicate data using Atom + RSS. You can build RESTful services that use PUT, DELETE, and other features of HTTP. You can use the Relay service as a broker for doing NAT traversal securely.
