Apache Axiom 1.0

Stefan Tilkov, May 3, 2006

Apache Axiom, the object model that sits in Axis 2’s core, has been released. I wasn’t aware that it could be used independently of Axis, definitely worth a look if only for its XOP/MTOM support.

Update: Elliotte Rusty Harold has some comments:

There are some interesting new ideas here, that should be explored further. However, as a library it’s clearly unsuitable for production use.
