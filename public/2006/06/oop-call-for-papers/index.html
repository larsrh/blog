OOP Call for Papers

Stefan Tilkov, Jun 16, 2006

Together with Dr. Gernot Starke, I’m happy to chair the 4-day SOA track at next year’s OOP 2007 conference which takes place from January 22-26 in Munich, Germany. Please see the call for papers for details about submitting an abstract. Note that we’re happy about English as well as German presentation proposals. The deadline is July 1st (!), so hurry up :-)
