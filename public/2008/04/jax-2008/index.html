JAX 2008

Stefan Tilkov, Apr 22, 2008

I'll be at JAX 2008 for the next three days. My colleague Phillip Ghadir will talk about "REST for SOA" today; Hartmut Wilms's topic tomorrow is "Heuristics for service design". Also tomorrow, I'll give an introductory Ruby talk on Wednesday, do a SOA overview together with Gernot Starke, and participate in a panel on Java EE vs. Rails. On Thursday, I'll talk about JSR 311; on Friday, I'll present on doing RESTful Web services with Ruby on Rails.

Be sure to drop by if you're a regular reader … I always like meeting both of them.
