Duck Typing for Web Services

Stefan Tilkov, Mar 27, 2007

As usual, I find it very easy to agree with Arjen Poutsma — he suggests to use duck typing for building Web services, and offers three tips:

  1. Don’t validate incoming messages!
  2. Use XPath!
  3. Don’t create stubs or skeletons!

Good advice. I’ve written similar things in the past; see also entry #6 here.
