MindTouch Dream: REST SDK &#38; Lightweight Standalone Server

Stefan Tilkov, Oct 10, 2007

Hartmut has written a brief overview of Dream, “a programming library and standalone server for developing and publishing RESTful web services. MindTouch Dream is on Open Source Project covered by the Gnu Lesser General Public License Agreement (LGPL). It is written in C# entirely and runs on Microsoft .NET 2.0 as well as Novell Mono 1.2.5.”
