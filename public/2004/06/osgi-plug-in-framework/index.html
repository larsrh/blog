OSGi Plug-in framework

Stefan Tilkov, Jun 15, 2004

Mark and Don Park both link to OSGi, a standard lightweight plug-in framework. Oscar is an open source implementation, and it seems that Eclipse 3’s plug-in architecture is based on OSGi as well.
