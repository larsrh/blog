Fighting Comment Spam

Stefan Tilkov, Nov 6, 2004

MT-DSBL, an MT 3.1 plugin from Brad Choate, blocks comments made from blacklisted IP addresses (e.g. through anonymous proxies). While there hasn’t been any visible comment spam on my blog for a few months (MT-Blacklist catches everything, but unfortunately has some false positives), going through comments and blacklisting is annoying work, and any way to avoid it is welcome.
