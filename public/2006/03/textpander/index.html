Textpander

Stefan Tilkov, Mar 5, 2006

Textpander is a nicely done and free utility for OS X that enables you to insert text snippets based on shortcuts. I used to use TypeIt4Me before, but somehow can’t seem to get it running on the Intel Mac. Peter Maurer’s other software is also worth checking out.
