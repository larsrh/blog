Open Source Voting Software

Stefan Tilkov, Nov 5, 2003

Sean McGrath writes about an Open Source voting system called Choice Plus. I remember from a discussion about a year ago with a potential German customer who built a system for use in German elections (not that I can recall it ever having been used) that they were actually forced to use only Open Source software by the state government.

On November 11, 2004 5:20 PM, Laird Popkin said:

If you’re a supporter of Open Source Voting, please help out the Open Voting Consortium. They’re not only development a software system that implements touscreen voting with voter verified paper ballots, they’re working with numerous states to advance the use of open source voting systems.
