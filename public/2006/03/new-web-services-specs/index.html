New Web Services Specs

Stefan Tilkov, Mar 2, 2006

Chris Ferris:

IBM, Microsoft, Oracle, SAP AG have published two new Web services specifications: SOAP 1.1 Binding for MTOM 1.0 and WSDL 1.1 Binding Extension for SOAP 1.2.

One might wonder why this has not been published through the W3C — but my guess is that the harmless reason is that doing anything for SOAP 1.1 or WSDL 1.1 is no longer on the official agenda.
