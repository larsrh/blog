Clojure 1.0

Stefan Tilkov, May 4, 2009

Rich Hickey:

After a sustained period of API stability and minimal bugs reports, I'm happy to announce the release of Clojure 1.0!
http://clojure.googlecode.com/files/clojure_1.0.0.zip

Clojure is a Lisp for the JVM, with an excellent set of core concepts, great support for concurrent programming, and superb Java integration. In my very humble opinion, it's the first time Lisp has a chance to become at least somewhat mainstream.
