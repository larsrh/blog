Apple iWork '06

Stefan Tilkov, Feb 2, 2006

I’m currently playing with Apple’s iWork ‘06, and while I don’t care that much for the new features in Keynote, I thought I’d give Pages another try. It’s actually a pretty nice application — it has a very clean and simple UI with none of the bloat of Word. While my main reason for using Word in the first place — interoperability and support for tracking changes — will remain, some features in Pages are really cool: after some googling around, I found out about the way to change a document’s language, and one of the available settings is “multilingual”, which allows you to write a typical non-English technical document with its weird language mix without getting every technical term underlined. I’m going to give it a try for some private writing.
