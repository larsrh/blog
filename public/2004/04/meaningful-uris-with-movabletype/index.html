Meaningful URIs with MovableType

Stefan Tilkov, Apr 18, 2004

I have changed the method for constructing weblog entry URIs, following this nice and detailed description by Jim Ray, to appear more meaningful. For example, what used to be

/blog/st/archives/000841.html

is now 

/blog/st/2004/04/the_hidden_impact_of_wsaddressing_on_soap.html

The old pages with the silly, number-based naming scheme are still there, but redirect to their successors automatically.
