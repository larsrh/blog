A Spreadsheet-based Web Interface?

Stefan Tilkov, Aug 16, 2007

Phil Windley wonders why it’s not easier to integrate data into spreadsheets. This makes me wonder: wouldn’t it be neat (and very straightforward to do) if you could just put up an Excel sheet to some server, and get at the data inside it via a Google Spreadsheet style API?

On August 27, 2007 6:21 AM, sumedha rubasinghe said:

WSO2 Data Services allow you to expose data in a Excel spreadsheet via a Web Service. Sounds interesting ? Visit http://www.wso2.org/blog/sumedha/2581 to find more on how to do it.

Additionally you can give a Web Service Interface to data stored on a RDBMS or CSV file using WSO2 Data Services.
