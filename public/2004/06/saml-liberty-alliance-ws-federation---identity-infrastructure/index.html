SAML, Liberty Alliance, WS-Federation - Identity Infrastructure

Stefan Tilkov, Jun 10, 2004

Via Phil Windley, I came across SourceID.org. This is an interesting project, with a number of interesting components. SourceID.Java is an open-source Java toolkit for enabling Liberty protocol v1.1 Single Sign-On and federated identity exchange; SourceID.NET is a open-source developer’s tool for integrating the Liberty Protocol (Phase I) into .NET web applications (C# and ASP.NET). There is also a Proof of Concept of the WS-Federation specification’s Passive (Brower) Profile, written in ASP.NET against the .NET Framework version 1.1.

Definitely something to watch …
