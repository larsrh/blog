Jettison 1.0-beta-1

Stefan Tilkov, Feb 3, 2007

Dan Diephouse:

I am happy to announce the availbility of Jettison 1.0-beta-1. Jettison is a StAX implementation which reads and writes JSON. It was designed for usage in CXF and XFire to allow easy creation of JSON services side by side with your XML services.
