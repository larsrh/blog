WSDL 2.0 Becomes Candidate Recommendation

Stefan Tilkov, Jan 11, 2006

WSDL 2.0 has achieved CR status — wow, that was fast: Start with WSDL 1.1, and a mere 5 years later, you’re almost done ;-)

The relevant docs are Primer (Part 0, recommended for getting an overview), Core Language and schema (Part 1), Adjuncts (Part 2), SOAP 1.2 binding schema, HTTP binding schema and SOAP 1.1 Binding.
