Go with Atom

Stefan Tilkov, Jul 5, 2006

I’m not sure anyone cares about my opinion in this particular discussion, but I agree 100% with James Robertson’s view (who references this post by DeWitt Clinton): If you need to provide content syndication features in any form, you should definitely go with Atom (which is an IETF standard), not RSS (which is, technically speaking, crap). Convert as needed.
