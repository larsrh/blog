Windows on a Mac Book Pro

Stefan Tilkov, Apr 9, 2006

For some reason that I can’t remember very clearly, I couldn’t resist trying out Boot Camp — and promptly ended up fucking up my system so thoroughly that I could no longer boot OS X :-( Ironically enough, Windows still started up nicely … but in the meantime, Parallels has released a beta of their virtualization solution for OS X on Intel, and I can happily report that it works like a charm:

I’m not yet sure about the performance, but without a doubt it’s significantly faster than Virtual PC ever was. I’ll report back once I have installed some applications …

In any case, I have no doubts either this, VMware’s or possibly a solution by Apple itself will ultimately turn an Intel Mac into the most useful machine one can buy.
