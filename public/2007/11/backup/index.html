Backup

Stefan Tilkov, Nov 26, 2007

Gernot Starke writes about his backup strategy — pretty neat. I second the recommendation for SuperDuper: Even though it’s only a front-end to ditto, it incorporates useful profiles, an incremental mode and has Apple’s recommended set of directories and files built in.
